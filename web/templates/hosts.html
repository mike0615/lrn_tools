Key-based Authentication (Recommended)

Generate a key pair on this server and copy the public key to each remote host:

ssh-keygen -t ed25519 -f ~/.ssh/lrn_tools_key
ssh-copy-id -i ~/.ssh/lrn_tools_key.pub user@remote-host

Then set Auth Type to Key and Key Path to ~/.ssh/lrn_tools_key.

lrn_tools on Remote Hosts

The tool scripts must exist on the remote host at the configured path. Clone the repo:

git clone https://github.com/mike0615/lrn_tools.git ~/dev/lrn_tools

Use the Test button to verify both SSH connectivity and lrn_tools presence.

Password Authentication

Requires sshpass on this server. Passwords are stored in plain text in ~/.lrn_tools/hosts.json (chmod 600). Use key auth when possible.

dnf install sshpass
